== Installation ==

1. Upload the `plugin` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to Settings>Penci Slider Options for configuration general.
1. Go to Penci Slider>Add New for create a slide.
1. Use shortcode to display slider everywhere you want.
